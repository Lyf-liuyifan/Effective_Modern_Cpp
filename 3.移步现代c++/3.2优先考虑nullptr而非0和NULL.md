# 3.2优先考虑nullptr而非0和NULL
0和NULL在有函数重载了整形的时候是不会调用形参类型是指针的函数的。

```
void f(int);        //三个f的重载函数
void f(bool);
void f(void*);

f(0);               //调用f(int)而不是f(void*)

f(NULL);            //可能不会被编译，一般来说调用f(int)，
                    //绝对不会调用f(void*)
```

**而nullptr对于任何指针类型都是指针而且不会有二义性问题出现。**
